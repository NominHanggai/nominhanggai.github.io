Quantum process tomography and the transfer tenor method for long-time non-Markovian dynamics

\@Cheet on 2022/04/05

Predicting long-time dynamics from short-time information is a promising way of studying how equilibrium is approached for an open quantum system starting from a non-equilibrium initial condition.

This post discusses one of the prediction methods for long-time open system dynamics, the transfer tensor method (TTM)1. The TTM is based on (a) the Nakajima-Zwanzig equation equation of an open quantum system [Eq. (1)], and (b) the assumption that the state of the open system at time $t$ is barely correlated with the state at $t'$ if $|t-t'|$ is greater than a certain threshold. Let’s start from the Nakajima-Zwanzig equation of an open system 2:
\begin{align} \frac{d}{d t} \hat{\rho}_{s}(t)=-\frac{i}{\hbar} \mathcal{L}_{s} \hat{\rho}_{s}(t)-\int_{0}^{t} d \tau\, \mathcal{K}(\tau) \hat{\rho}_{s}(t-\tau) \tag{1}. \end{align}
where $\mathcal{L}_{s}$ is the Liouville operator corresponding to the system part of the system-bath Hamiltonian. The memory kernel $\mathcal{K}(\tau)$ measures how correlated a state $\rho(t)$ is to a previous state $\rho(t-\tau)$. This this equation can be discretized on a time grid $(0, \delta, 2\delta, \ldots, N\delta)$ as follows:
\begin{align} \hat{\rho}_{s}(t_{N}) = (1- \frac{i}{\hbar} \mathcal{L}_{s}\delta)\rho(t_{N-1}) -\sum_{n=1}^{N} \mathcal{K}(n\delta)\delta^2\underbrace{\rho(N\delta-n\delta)}_{\rho[(N-n)\delta]} \tag{2}. \end{align}

Figure 1. The discretized integral in Eq. (1).

This means, if we know that memory kernel $\mathcal{K}(t)$ decays to zero after a certain time $t_\mathrm{max}$, we can use the memory kernel and reduced density matrices before $t_\mathrm{max}$ to calculate the reduced density matrix to as long a time as we want. For example, if $\mathcal{K}(t>10\delta)=0$, we can use a sequence of initial density matrices $\hat{\rho(0)},\ldots,\hat{\rho(9\delta)}$ and $\mathcal{K}(\delta),\ldots,\mathcal{K}(10\delta)$ to calculate $\hat{\rho}(11\delta)$. Again, based on $\hat{\rho(\delta)},\ldots,\hat{\rho(11\delta)}$ (just obtained in the last step) and $\mathcal{K}(\delta),\ldots,\mathcal{K}(10\delta)$ we can obtain $\hat{\rho}(12\delta)$. This calculation can go on and on until we get the density matrix at a desired long enough time.

The question is how to obtain the initial sequence$\hat{\rho_{1}},\ldots,\hat{\rho_{10}}$ and $\mathcal{K}(t_{0}),\ldots,\mathcal{K}(t_{10})$. The density matrix part can be easily to obtain by simulating the open system with an desired initial condition. The memory kernel is a bit more difficult to obtain. To calculate $\mathcal{K}(t_{n})$, we first note that Eq. (2) can be rewritten as
\begin{align} \hat{\rho}_{s}(t_{1}) = \mathcal{E}_{1}\hat{\rho}_{s}(t_{0}) \equiv (1- \frac{i}{\hbar} \mathcal{L}_{s})\hat{\rho}_{s}(t_{0}) - \mathcal{K}_{t_0}\hat{\rho}_{t_0}\\ \hat{\rho}_{s}(t_{2}) = \mathcal{E}_{1}\hat{\rho}_{s}(t_{0}) \equiv (1- \frac{i}{\hbar} \mathcal{L}_{s})\hat{\rho}_{s}(t_{0}) - \mathcal{K}_{t_0}\hat{\rho}_{t_0}\\ (1- \frac{i}{\hbar} \mathcal{L}_{s})\rho(t_{N-1}) -\sum_{n=0}^{N-1} \mathcal{K}(t_{k})\underbrace{\rho(t_N-t_n)}_{\rho(t_{N-n})} \tag{2}. \end{align}
*under construction
