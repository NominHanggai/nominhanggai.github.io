Миний аав улаанбаатарт байна

\@Cheet on 2022/01/26

Миний аав Улаанбаатарт байна (миний хүү аавыгаан их санасан юм бай)
Миний аав уужам тайбан сэдьхэлтэй (уужуу тайван сэтгэлтэй шүү )
Миний ааваа морин хуураар татаад (морин хуураа татаад гэж болж бай)
Миний аав сайн суугаарай! (сайн байна миний хүү)

Миний хүү Бээжин хотод байна (миний аав миний аав дуулж байна )
Миний хүү хөөрхэн цовоо ааштай (намайг хөөрхөн гэж байна, би тийм хөөрхөн гэж үү?)
Миний хүү тэбтэг хуураар тоглоод (би хуураа тоглоогvй )
Миний хүү сайн суугаа даа! (би тун сайн байна )
